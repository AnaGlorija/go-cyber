| /bandwidth/load                       |                      | get bandwidth load                         |
| /bandwidth/price                      |                      | get bandwidth price                        |
| /bandwidth/desirable                  |                      | get desirable bandwidth                    |
| /bandwidth/account/{address}          | {address}            | get bandwidth of give account address      |
